Schematic:

The schematic for this assignment consists of two LED pathways each starting at a ~ pin (~6 & ~9) on the arduino, then going through a 330 Ohm resistor, LED, and then ground. Additionally, there is also a photoresistor pathway that is connected to the 5V power on the arduino, pin A0, ground, a 10k resistor, and a photoresistor.


A 10k resistor was used for the photoresistor since the photoresistor has a large range so the 10k resistor is able to make the circuit run safely while not effecting the voltage signiticantly. For the LED resistors, the calculation for A1 Blink were referenced.

Firmware:

Here is the code used for A3: Input Output. This code initializes pins 6 and 9 then reads pin A0 and fades the LEDS based on the pin A0 reading.


Note: Serial.println was used to print the values from the photoresistor via analogRead using pin A0. This process showed the values ranged from around 400 to 800 as seen in the gif below. Thus, the values used were contstrained to 400-800 so 600 would be the perfect midpoint and switch which LED is on.


/*

  A3 Input Output

*/


void setup() {

  // initalizes pin 6 to fade yellow LED

  pinMode(6, OUTPUT);

  // initalizes pin 9 to fade blue LED

  pinMode(9, OUTPUT);

  // initalize serial communications at 9600 bps

  Serial.begin(9600);

}


void loop() {

  // initializes variable which is the value of the photoresistor:

  int val = analogRead(A0);

  // prints value of analogRead --> shows that its around 400 to 800

  Serial.println(val);

  // limits range of sensor values to between 400 and 800

  int sensVal = constrain(val, 400, 800);

  // connects photoresistor values to led brightness values 

  int outVal = map(sensVal, 400, 800, 0, 255);


  // if statement for when sensory value is above 600

  if (sensVal >= 600) {

    // makes yellow led brightness the photoresistor value

    analogWrite(6, outVal);

    // tuns off blue led

    digitalWrite(9, LOW);

  // else for when sensory value is below 600

  } else {

    // makes blue led brightness the photoresistor value

    analogWrite(9, outVal);

    // tuns off yellow led

    digitalWrite(6, LOW);

  }

}
